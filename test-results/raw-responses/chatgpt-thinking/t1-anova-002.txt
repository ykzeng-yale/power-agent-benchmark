For a **one-way ANOVA** with **4 dosage groups** (k = 4), Cohen’s **f = 0.30**, **α = 0.05**, and **80% power**, we solve using the noncentral F distribution:

* (df_1 = k-1 = 3)
* (df_2 = N-k)
* Noncentrality ( \lambda = N f^2 )

Searching for the smallest **equal** group size (n) (so (N=4n)) that achieves ≥80% power gives:

* **(n = 32) patients per dosage group**
* Total **(N = 128)**
* Achieved power ≈ **0.81**

So you need **32 patients per dosage level** (≈ **128 total**).
