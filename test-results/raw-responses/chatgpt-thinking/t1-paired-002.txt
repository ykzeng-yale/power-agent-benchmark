This is a **paired (preâ€“post) design**, so the relevant analysis is a **one-sample / paired t-test** on the **within-person change**.

Let

* Expected mean change (loss): ( \Delta = 3) kg
* SD of the within-person changes: ( \sigma_d = 5) kg
* Two-sided ( \alpha = 0.05 \Rightarrow z_{1-\alpha/2} = z_{0.975} = 1.96)
* Power (= 90% \Rightarrow \beta = 0.10 \Rightarrow z_{1-\beta} = z_{0.90} = 1.2816)

Approximate sample size for a paired t-test (normal approximation):

[
n ;=;\left(\frac{(z_{1-\alpha/2}+z_{1-\beta}),\sigma_d}{\Delta}\right)^2
]

Plug in:

[
n = \left(\frac{(1.96+1.2816)\cdot 5}{3}\right)^2
= \left(\frac{3.2416\cdot 5}{3}\right)^2
= \left(5.4027\right)^2
\approx 29.19
]

So you need **(n = 30)** participants (round up).

Practical note: if you expect dropouts, inflate. For example, with **15% dropout**, target (30/0.85 \approx 35.3 \Rightarrow) **36 participants** enrolled.
