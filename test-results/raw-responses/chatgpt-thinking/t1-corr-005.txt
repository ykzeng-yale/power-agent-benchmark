To power a **one-sided test for a Pearson correlation** ( (H_0:\rho=0) vs (H_1:\rho>0) ) using the Fisher *z* transform:

[
n ;=;\left(\frac{z_{1-\alpha}+z_{1-\beta}}{\operatorname{atanh}(r)}\right)^2 + 3
]

Given:

* Expected correlation: (r = 0.20)
* One-sided (\alpha = 0.025 \Rightarrow z_{1-\alpha}=z_{0.975}\approx 1.96)
* Power (=0.85 \Rightarrow \beta=0.15,; z_{1-\beta}=z_{0.85}\approx 1.036)
* (\operatorname{atanh}(0.2)=\tfrac12\ln\left(\frac{1+0.2}{1-0.2}\right)\approx 0.2027)

Compute:

[
n \approx \left(\frac{1.96+1.036}{0.2027}\right)^2 + 3
= \left(\frac{2.996}{0.2027}\right)^2 + 3
\approx (14.78)^2 + 3
\approx 218.4 + 3
\approx 221.4
]

**Required sample size:** **(n = 222)** patients (rounding up).
