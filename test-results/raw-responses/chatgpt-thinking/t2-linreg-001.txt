To power the **overall multiple regression F-test** (testing whether the set of predictors explains any variance), you can use Cohenâ€™s (f^2) effect size for (R^2):

### 1) Convert the target (R^2) to (f^2)

[
f^2=\frac{R^2}{1-R^2}=\frac{0.10}{0.90}=0.1111
]

### 2) Set up the F test

* Number of predictors: (p=5)
* Numerator df: (\text{df}_1=p=5)
* Denominator df: (\text{df}_2 = N - p - 1 = N-6)
* For this global test, the noncentrality parameter is:
  [
  \lambda = f^2 \cdot (N-p-1)= f^2 \cdot (N-6)
  ]
  Power is:
  [
  \Pr\left(F_{5,,N-6,;\lambda} > F_{1-\alpha;,5,,N-6}\right)
  ]

### 3) Solve for (N) giving 80% power at (\alpha=0.05)

Plugging in (f^2=0.1111), (p=5), (\alpha=0.05) and solving the noncentral F power equation gives:

[
\boxed{N \approx 127 \text{ total subjects}}
]

So you need **about 127 participants** to have **80% power** to detect an overall (R^2 = 0.10) with **5 predictors** at **(\alpha=0.05)**.

Practical note: if you expect missing outcomes/covariates, inflate (e.g., 10% missing (\Rightarrow 127/0.90 \approx 142)).
