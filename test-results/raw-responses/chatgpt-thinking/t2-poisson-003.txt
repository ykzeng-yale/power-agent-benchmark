Assume each subject has the same follow-up/exposure time in both arms, and counts follow a **negative binomial (NB2)** model with mean (\mu) and dispersion (size) (k) so that

[
\mathrm{Var}(Y)=\mu+\frac{\mu^2}{k}.
]

You want to compare **mean counts**: control (\mu_0=5), treatment (\mu_1=4), with (k=2). The effect is a rate ratio (mean ratio)

[
RR=\frac{\mu_1}{\mu_0}=\frac{4}{5}=0.8,\quad \Delta=\log(RR)=\log(0.8)=-0.2231.
]

### Approximate variance and sample size (Wald test on (\log RR))

Using the delta method on the sample means, for one group of size (n),

[
\mathrm{Var}\big(\log(\bar Y)\big)\approx \frac{\mathrm{Var}(\bar Y)}{\mu^2}
= \frac{\mu+\mu^2/k}{n\mu^2}
= \frac{1}{n\mu}+\frac{1}{nk}.
]

So with equal (n) per group, the variance of (\log(\bar Y_1)-\log(\bar Y_0)=\log(RR)) is

[
\mathrm{Var}(\log RR)\approx \frac{1}{n}\left(\frac{1}{\mu_1}+\frac{1}{\mu_0}+\frac{2}{k}\right).
]

Powering a two-sided test at (\alpha=0.05) with 80% power:

[
n ;=; \frac{(z_{1-\alpha/2}+z_{1-\beta})^2\left(\frac{1}{\mu_1}+\frac{1}{\mu_0}+\frac{2}{k}\right)}{(\log RR)^2}.
]

Plug in:

* (z_{1-\alpha/2}=1.96), (z_{1-\beta}=0.84)  (\Rightarrow (1.96+0.84)^2=2.80^2=7.84)
* (\frac{1}{\mu_1}+\frac{1}{\mu_0}+\frac{2}{k}=\frac14+\frac15+\frac{2}{2}=0.25+0.20+1.00=1.45)
* ((\log 0.8)^2 = 0.2231^2 \approx 0.0498)

[
n \approx \frac{7.84\times 1.45}{0.0498}\approx \frac{11.37}{0.0498}\approx 228.5
]

### Result

**You need about (\boxed{229}) participants per group** (so **(\boxed{458}) total**) for 80% power at (\alpha=0.05).

If you expect dropout or unequal follow-up time, inflate (n) accordingly (e.g., divide by expected retention).
