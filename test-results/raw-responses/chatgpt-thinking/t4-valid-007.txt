To plan sample size for **precision of the C-statistic (AUC)**, we can treat the AUC estimate as approximately normal and require the **95% CI width ≤ 0.10**.

## 1) Translate CI width requirement into an SE target

For a Wald-type CI:
[
\text{CI width} \approx 2 z_{0.975}, SE(\widehat{AUC})
]
You want width ≤ 0.10, so:
[
SE(\widehat{AUC}) \le \frac{0.10}{2\times 1.96}=0.02551
]

## 2) Use an AUC variance approximation (Hanley–McNeil)

Let (A=0.75) be the *expected* AUC in the validation population, with:

* (n_1) = number of events (cases)
* (n_0) = number of non-events (controls)

Hanley–McNeil variance:
[
\mathrm{Var}(\widehat{A})=
\frac{
A(1-A) + (n_1-1)(Q_1-A^2) + (n_0-1)(Q_2-A^2)
}{n_1 n_0}
]
where
[
Q_1=\frac{A}{2-A},\qquad Q_2=\frac{2A^2}{1+A}
]

For (A=0.75):

* (Q_1 = 0.75/(1.25)=0.60)
* (Q_2 = 2(0.75^2)/(1.75)=0.642857)

## 3) Plug in prevalence to link (n_1,n_0) to total (N)

Outcome prevalence (p=0.12), so in expectation:
[
n_1 \approx 0.12N,\qquad n_0 \approx 0.88N
]

Solving for the smallest integer (N) such that
[
2\times 1.96 \times \sqrt{\mathrm{Var}(\widehat{A})}\le 0.10
]
gives:

* **Total sample size:** **(N \approx 1{,}096)**
* **Expected events:** (0.12\times 1096 \approx 132)
* **Expected non-events:** (964)

This yields a 95% CI width for the AUC of about **0.0998** (meets the ≤0.10 target).

## Recommendation

Plan for **~1,100 patients total** (≈**132 events**) **minimum**, then inflate for:

* missing outcome data,
* loss to follow-up,
* exclusions.
