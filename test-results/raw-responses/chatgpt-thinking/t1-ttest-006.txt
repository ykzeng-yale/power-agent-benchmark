For a **two‐arm, equal‐allocation, two‐sample t-test** with standardized effect size (d=\frac{\mu_1-\mu_2}{\sigma}=0.8), **two-sided** (\alpha=0.001), and **power** (1-\beta=0.99):

### 1) Common planning formula (normal approximation)

[
n \approx \frac{2\left(z_{1-\alpha/2}+z_{1-\beta}\right)^2}{d^2}
]

* (z_{1-\alpha/2} = z_{0.9995} \approx 3.2905)
* (z_{1-\beta} = z_{0.99} \approx 2.3263)

[
n \approx \frac{2(3.2905+2.3263)^2}{0.8^2}
= \frac{2(5.6168)^2}{0.64}
\approx \frac{2\cdot 31.55}{0.64}
\approx 98.6
]
So this approximation gives **~99 participants per arm**.

### 2) Using the exact t-critical value (more accurate)

If you use the **t distribution** critical value (recommended in confirmatory settings), you need a bit more because the critical value depends on degrees of freedom:

* Solving with noncentral t power gives **(n = 102) per arm** to achieve (\ge 0.99) power at two-sided (\alpha=0.001).

## Answer

**Plan for 102 participants per treatment arm** (≈ **204 total**), *before* adding any inflation for dropout/nonadherence.

If you expect attrition, inflate: (n_\text{adj} = \frac{102}{1-\text{dropout rate}}) (e.g., 10% dropout → (102/0.9 \approx 114) per arm).
