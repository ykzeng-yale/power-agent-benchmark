Assuming a **two-arm cluster randomized trial**, **two-sided α = 0.05**, equal cluster sizes, and no covariate adjustment:

### 1) Design effect

[
DE = 1 + (m-1)\rho = 1 + (30-1)\cdot 0.08 = 1 + 2.32 = 3.32
]

### 2) Standard error factor for the standardized mean difference

With (J=15) schools/arm and (m=30) students/school:
[
\sqrt{\frac{2,DE}{Jm}}=\sqrt{\frac{2\cdot 3.32}{15\cdot 30}}
=\sqrt{\frac{6.64}{450}} \approx 0.12147
]

### 3) Critical values using cluster df

A common approximation uses (df = 2J-2 = 28):

* (t_{0.975,28} \approx 2.048) (two-sided α=0.05)
* (t_{0.80,28} \approx 0.855) (80% power ⇒ (1-\beta=0.80))

### 4) Minimum detectable effect size (MDE)

[
d_{\min} \approx \left(t_{0.975,28}+t_{0.80,28}\right)\sqrt{\frac{2,DE}{Jm}}
= (2.048+0.855)\cdot 0.12147 \approx 0.353
]

**Minimum detectable Cohen’s d ≈ 0.35** (about a *small-to-moderate* effect).

If you plan to adjust for strong baseline covariates (e.g., pretest scores), the MDE can drop meaningfully; if cluster sizes vary a lot, it can increase.
