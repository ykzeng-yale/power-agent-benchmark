For a **2-arm non-inferiority trial with a binary response**, an often-used planning approximation (Wald/normal) is

[
n_{\text{per group}}
;\approx;
\frac{(z_{1-\alpha}+z_{1-\beta})^2,[p_C(1-p_C)+p_T(1-p_T)]}{\Delta^2}
]

Where

* (p_C) = control response rate (standard)
* (p_T) = true new-treatment response rate under the alternative (here “similar”, so use (p_T=p_C=0.70))
* (\Delta) = **non-inferiority margin** on the absolute difference scale (here 0.10)
* one-sided (\alpha=0.025 \Rightarrow z_{1-\alpha}=1.96)
* power (=90% \Rightarrow \beta=0.10 \Rightarrow z_{1-\beta}=1.2816)

Plug in:

* (p_C(1-p_C)=0.7\times 0.3=0.21)
* (p_T(1-p_T)=0.21)
* Sum (=0.42)
* (z_{1-\alpha}+z_{1-\beta}=1.96+1.2816=3.2416)
* Square: (3.2416^2 \approx 10.512)

[
n \approx \frac{10.512 \times 0.42}{0.10^2}
= \frac{4.415}{0.01}
= 441.5
]

So you need **about (\lceil 441.5\rceil = 442) patients per group** (≈ **884 total**).

If you expect dropout/nonadherence, inflate: e.g., **10% loss → (442/0.9 \approx 492) per group**.
