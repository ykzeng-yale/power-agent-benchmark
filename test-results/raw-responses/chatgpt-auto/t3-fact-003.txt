Treat the **linear dose–response trend** as a **1 df** effect tested in a fixed-effects ANOVA/GLM for the **6-cell (3×2) design**.

### Model + degrees of freedom

* Cells: (g = 3\times 2 = 6)
* Numerator df for the linear trend: (u = 1)
* Error df: (v = N - g = N - 6)

### Effect size to noncentrality

Using Cohen’s (f) for the tested effect in a GLM/F test, the noncentrality parameter is:
[
\lambda = f^2,N
]
Here (f=0.25 \Rightarrow f^2 = 0.0625), so (\lambda = 0.0625,N).

### Power condition

Power is:
[
\text{Power} = P\left(F_{u,v,\lambda} > F_{1-\alpha;u,v}\right),
\quad \alpha=0.05
]
Solving this gives **minimum (N \approx 128)** for 80% power.

### Practical balanced allocation

Because you usually want equal (n) per cell:

* (N=128) implies (128/6 \approx 21.33) per cell (not balanced).
* Next balanced total is **(22) per cell**:
  [
  N = 6\times 22 = 132
  ]
  This yields power slightly above 0.80.

**Answer:**

* **Minimum total (N \approx 128)** (mathematically).
* **Recommended (balanced): (N = 132) total** = **22 per cell**.
