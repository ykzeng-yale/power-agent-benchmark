To power a **test of correlation** (typically Pearson) with

* Expected true correlation: ( \rho = 0.15 )
* Two-sided ( \alpha = 0.05 )
* Power ( 1-\beta = 0.80 )

a standard approach uses the **Fisher (z)-transform**.

## Step 1) Fisher transform of the expected correlation

[
\delta ;=; \operatorname{arctanh}(\rho)
;=; \frac12 \ln!\left(\frac{1+\rho}{1-\rho}\right)
]
For (\rho=0.15):
[
\delta = \operatorname{arctanh}(0.15) \approx 0.15114
]

## Step 2) Critical values

Two-sided (\alpha=0.05 \Rightarrow z_{1-\alpha/2} = z_{0.975} \approx 1.95996)

Power (0.80 \Rightarrow z_{1-\beta} = z_{0.80} \approx 0.84162)

## Step 3) Sample size formula

For testing (H_0:\rho=0) vs (H_1:\rho\neq 0), an approximation is:
[
n \approx \left(\frac{z_{1-\alpha/2}+z_{1-\beta}}{\delta}\right)^2 + 3
]

Plug in:
[
n \approx \left(\frac{1.95996 + 0.84162}{0.15114}\right)^2 + 3
\approx 346.59
]

## Result

[
\boxed{n = 347 \text{ patients (round up)}}
]
