Using the standard LMM information-matrix approximation (Wald z test), with **complete data**, **40 per group** (80 total), **times = 0,1,2,3,4,5**, and **random intercept/slope independent** (no intercept–slope covariance specified):

### Model

[
y_{ij}=\beta_0+\beta_1 t_{j}+\beta_2,\text{group}*i+\beta_3(\text{group}*i\cdot t_j)+b*{0i}+b*{1i}t_j+\varepsilon_{ij}
]

* Test: (H_0:\beta_3=0) vs (H_1:\beta_3=0.1) (two-sided (\alpha=0.05))
* SDs: (\sigma_{b0}=1.0), (\sigma_{b1}=0.2), (\sigma_e=1.0)

### Key calculation (variance of the interaction slope estimate)

For each subject, the marginal covariance is
[
V = \sigma_{b0}^2 \mathbf{1}\mathbf{1}^\top + \sigma_{b1}^2, tt^\top + \sigma_e^2 I_6
]
and
[
\mathrm{Var}(\hat\beta)=\left(X^\top V^{-1}X\right)^{-1}
]
summed over subjects in each group.

This gives:

* (\mathrm{Var}(\hat\beta_3)\approx 0.004857)
* (\mathrm{SE}(\hat\beta_3)=\sqrt{0.004857}\approx 0.0697)

So the noncentrality for the true effect (0.1) is
[
\text{ncp}=\frac{0.1}{0.0697}\approx 1.435
]

### Power (two-sided, (\alpha=0.05))

With (z_{0.975}=1.96),
[
\text{Power}=P(|Z|>1.96 \mid Z\sim N(1.435,1)) \approx 0.30
]

## ✅ Result

**Power (\approx 0.30) (about 30%)** to detect a **slope difference of 0.1 per time unit** with **40 subjects per group**.
