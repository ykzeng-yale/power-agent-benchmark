For a **one-way ANOVA (omnibus F test)** with **3 equal-sized groups**, effect size **Cohen’s (f=0.25)**, **(\alpha=0.05)**, and **80% power**, we solve using the noncentral F distribution.

### Model and formulas

* Number of groups: (k=3)
* Total sample size: (N = k,n) where (n) = subjects per group
* Degrees of freedom:

  * (df_1 = k-1 = 2)
  * (df_2 = N-k = 3n-3)
* Noncentrality parameter for the omnibus ANOVA:
  [
  \lambda = f^2 , N
  ]
  Here (f^2 = 0.25^2 = 0.0625), so (\lambda = 0.0625 \cdot N).
* Critical value:
  [
  F_{\text{crit}} = F_{1-\alpha}(df_1, df_2)
  ]
* Power:
  [
  \text{Power} = P!\left(F_{df_1,df_2,\lambda} > F_{\text{crit}}\right)
  ]

### Result (smallest (n) achieving ≥ 0.80 power)

Checking (n) values:

* (n=52) per group → power (\approx 0.7967) (just under 0.80)
* **(n=53) per group → power (\approx 0.8049)** (meets 0.80)

✅ **Answer: ( \boxed{53 \text{ subjects per group}} )**
That’s a total of **(N = 3 \times 53 = \boxed{159})** participants.

### Practical note

If you expect attrition, inflate: e.g., for 10% dropout, target (53/0.90 \approx 58.9) → **59 per group**.
