Using Riley’s *shrinkage* criterion (target uniform shrinkage factor (S=0.90)), the required development sample size for a regression prediction model is computed from (Figure 3): 

[
n ;=; \frac{P}{(S-1),\ln!\left(1-\frac{R^2}{S}\right)}
]
where

* (P) = number of candidate predictor parameters (here 30),
* (R^2) = anticipated model fit (here 0.35 for a continuous outcome),
* (S) = target shrinkage (here 0.90). 

Plug in:

* (S-1 = -0.10)
* (\frac{R^2}{S} = \frac{0.35}{0.90} = 0.3889)
* (1-\frac{R^2}{S} = 0.6111)
* (\ln(0.6111) \approx -0.4925)

So,

[
n ;=; \frac{30}{(-0.10)(-0.4925)} \approx \frac{30}{0.04925} \approx 609.2
]

**Required sample size (round up): (\boxed{610}) participants.**

*(Note: this meets the shrinkage criterion; Riley’s “full” continuous-outcome guidance also checks other criteria (e.g., residual variance precision), which can sometimes push (n) higher.)*
