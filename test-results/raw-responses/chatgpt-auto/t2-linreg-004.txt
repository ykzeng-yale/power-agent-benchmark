For a multiple linear regression where you plan to test the **overall model** (i.e., (H_0: R^2 = 0)) with:

* number of predictors (p = 10)
* Cohen’s (f^2 = 0.15) (medium)
* power (1-\beta = 0.85)
* significance level (\alpha = 0.01)

the required sample size comes from the **noncentral F test** for multiple regression.

## Step-by-step

### 1) Convert the setup to the F-test degrees of freedom

For the overall regression F-test:

* Numerator df: (u = p = 10)
* Denominator df: (v = N - p - 1 = N - 11)

### 2) Noncentrality parameter

For this test, the noncentrality parameter is:

[
\lambda = f^2 \times N
]

So here:

[
\lambda = 0.15N
]

### 3) Critical value at (\alpha=0.01)

[
F_{\text{crit}} = F^{-1}(1-\alpha; u, v)
]

### 4) Power condition

Power is:

[
\text{Power} = P\left(F_{u,v,\lambda} > F_{\text{crit}}\right)
]

We increase (N) until this probability is at least 0.85.

## Result

The **smallest (N)** that achieves at least **85% power** is:

[
\boxed{N = 174}
]

(Here, (v = 174-11 = 163), and the resulting power is about **0.852**.)

## Practical note

* This is the sample size for the **complete-case analyzed dataset**. If you expect missingness/attrition, inflate: e.g., with 10% loss, target (174/0.90 \approx 194).
