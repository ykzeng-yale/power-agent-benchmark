Using a standard **cluster-randomized Poisson rate** approximation (equal follow-up, equal cluster sizes), you have:

* Clusters per arm: (K=20)
* Subjects per cluster: (m=25)  (\Rightarrow) persons per arm (=Km=500)
* ICC: (\rho=0.03)
* Control rate: (\lambda_0 = 4) events / person-year
* Target rate ratio: (RR=1.25 \Rightarrow \lambda_1=5) events / person-year
* Two-sided (\alpha=0.05 \Rightarrow z_{1-\alpha/2}=1.96)

## 1) Design effect

[
DE = 1+(m-1)\rho = 1+24(0.03)=1.72
]

## 2) Expected total events (assuming **1 person-year per subject**)

Per arm person-years (=500).

[
E_0 = 500\cdot 4 = 2000,\qquad
E_1 = 500\cdot 5 = 2500
]

## 3) SE for (\log(RR)) with clustering

For independent Poisson counts, (\mathrm{Var}(\log RR)\approx \frac{1}{E_1}+\frac{1}{E_0}).
With clustering, inflate by (DE):

[
\mathrm{Var}(\log RR)\approx DE\left(\frac{1}{E_1}+\frac{1}{E_0}\right)
]
[
SE \approx \sqrt{1.72\left(\frac{1}{2500}+\frac{1}{2000}\right)} \approx 0.0393
]

## 4) Power

[
\delta=\log(1.25)=0.2231,\qquad
Z = \frac{\delta}{SE}\approx \frac{0.2231}{0.0393}\approx 5.67
]

Two-sided power (normal approximation):
[
\text{Power} \approx \Phi(Z-1.96) \approx \Phi(3.71) \approx 0.9999
]

### **Answer**

**Power (\approx 99.99%)** (essentially ~100%) at (\alpha=0.05), **assuming ~1 person-year follow-up per subject**.

Because this is a *rate* outcome, power depends strongly on total person-time. If average follow-up were:

* **0.25 PY** per subject: power (\approx 80.9%)
* **0.1 PY** per subject: power (\approx 43.4%)
