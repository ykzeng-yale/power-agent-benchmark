Using the same setup as before (event prevalence (p=0.22), anticipated (C\text{-stat}=0.82), LP (\sim N(-1.75,,2.16^2))), we compute the **minimum total validation (N)** to meet **all** precision targets and then take the **largest** (N) required across them.

For any 95% CI width target (W), the corresponding SE target is
[
\text{SE}_\text{target}=\frac{W}{2\times 1.96}=\frac{W}{3.92}.
]

---

## 1) C-statistic CI width (\le 0.10)

Target SE:
[
\text{SE}_C \le \frac{0.10}{3.92}=0.02551.
]

Using the Hanleyâ€“McNeil large-sample variance for AUC with (n_1) events and (n_0) non-events:
[
\mathrm{Var}(\hat A)=\frac{A(1-A)+(n_1-1)(Q_1-A^2)+(n_0-1)(Q_2-A^2)}{n_1n_0},
]
where
[
Q_1=\frac{A}{2-A},\qquad Q_2=\frac{2A^2}{1+A}.
]
With (A=0.82): (Q_1\approx 0.6949,; Q_2\approx 0.7389).
Let (n_1=0.22N,; n_0=0.78N). Solving gives:

* **(N \approx 510)** total meets width (\le 0.10).

So: (\boxed{N_C \approx 510}).

---

## 2) Calibration slope CI width (\le 0.30)

Target SE:
[
\text{SE}_{\text{slope}} \le \frac{0.30}{3.92}=0.07653.
]

For the calibration model (\text{logit}(Y)=\alpha+\beta\cdot \text{LP}) (with intercept included), Fisher information implies:
[
\mathrm{Var}(\hat\beta)=\frac{\left[\left(E{w\cdot \mathbf{x}\mathbf{x}^\top}\right)^{-1}\right]_{22}}{N},
\quad \text{where } \mathbf{x}=(1,\text{LP}),; w=p(1-p),; p=\text{expit}(\alpha+\text{LP}).
]

We choose (\alpha) so that the marginal prevalence matches 0.22 under LP (\sim N(-1.75,2.16^2)), which yields (\alpha\approx -0.392). Under this distribution, the resulting information calculation gives approximately:
[
\text{SE}(\hat\beta)\approx \sqrt{\frac{5.174}{N}}.
]
Set (\sqrt{5.174/N}\le 0.07653\Rightarrow N \ge 883.4).

So: (\boxed{N_{\text{slope}} \approx 884}).

---

## 3) O/E ratio CI width (\le 0.30)

A common large-sample CI for O/E uses the log scale:
[
\log(\text{O/E}) \pm 1.96\sqrt{\frac{1}{O}},
]
so
[
\text{width on log scale} \approx 2\times 1.96\sqrt{\frac{1}{O}}=3.92\sqrt{\frac{1}{O}}.
]
Target width (\le 0.30) implies:
[
3.92\sqrt{\frac{1}{O}}\le 0.30
;\Rightarrow;
\sqrt{\frac{1}{O}}\le 0.07653
;\Rightarrow;
O \ge \frac{1}{0.07653^2}\approx 170.8.
]
So need at least **171 events**. With prevalence 0.22:
[
N \ge \frac{171}{0.22}=777.3 \Rightarrow 778.
]

So: (\boxed{N_{O/E} \approx 778}).
