For a balanced **2×2 factorial** with **independent groups** (n per cell) and a continuous outcome, the **interaction** is the “difference-in-differences” contrast:

[
(\mu_{11}-\mu_{10})-(\mu_{01}-\mu_{00})
]

If each cell has size (n) and the common SD is (\sigma), then

* (\text{Var}(\bar Y_{ij})=\sigma^2/n)
* The interaction contrast uses **4 cell means**, so
  [
  \text{SE}(\widehat{\Delta\Delta})=\sqrt{\frac{4\sigma^2}{n}}=\frac{2\sigma}{\sqrt{n}}
  ]

You defined the standardized interaction effect as
[
d = \frac{\Delta\Delta}{\sigma}=0.3
]
so under the alternative, the (approximate) z-statistic mean is
[
\frac{\Delta\Delta}{\text{SE}} = \frac{d\sigma}{2\sigma/\sqrt{n}}=\frac{d\sqrt{n}}{2}.
]

For **two-sided** (\alpha=0.05) and **power** (1-\beta=0.80):

* (z_{1-\alpha/2}=1.96)
* (z_{1-\beta}=0.84)

A standard planning approximation is:
[
\frac{d\sqrt{n}}{2} \approx z_{1-\alpha/2}+z_{1-\beta}
]

Solve for (n):
[
n \approx \left(\frac{2(z_{1-\alpha/2}+z_{1-\beta})}{d}\right)^2
= \left(\frac{2(1.96+0.84)}{0.3}\right)^2
= \left(\frac{5.60}{0.3}\right)^2
\approx 348.8
]

### Result

* **(n \approx 349) per cell**
* **Total (N \approx 4\times 349 = 1{,}396)**

(Those main effects (d=0.5) don’t reduce the required (n) for the interaction test; the interaction’s effect size and the contrast SE drive this.)
