You can treat this as a **two-sample comparison of means** (equal allocation), using Cohen’s standardized effect size (d).

### Setup

* Expected effect size: (d = 0.3) (small)
* One-sided (\alpha = 0.05 \Rightarrow z_{1-\alpha} = z_{0.95} \approx 1.645)
* Power (= 0.80 \Rightarrow \beta = 0.20 \Rightarrow z_{1-\beta} = z_{0.80} \approx 0.842)

### Sample size formula (per arm)

For equal-sized groups in a two-sample z/t approximation:

[
n ;=; \frac{2\left(z_{1-\alpha}+z_{1-\beta}\right)^2}{d^2}
]

Plug in:

1. Add the z-scores:
   [
   z_{1-\alpha}+z_{1-\beta} = 1.645 + 0.842 = 2.487
   ]

2. Square:
   [
   (2.487)^2 \approx 6.187
   ]

3. Multiply by 2:
   [
   2 \times 6.187 = 12.374
   ]

4. Divide by (d^2 = 0.3^2 = 0.09):
   [
   n = \frac{12.374}{0.09} \approx 137.49
   ]

### Result

* **Round up** → **(n = 138) patients per arm** (about **276 total**).

### Practical note (dropout inflation)

If you expect, say, **10% dropout**:
[
n_{\text{inflated}} = \frac{138}{0.90} \approx 153.3 \Rightarrow 154 \text{ per arm}
]

So: **138 per arm** (ideal), or **~154 per arm** with 10% attrition.
