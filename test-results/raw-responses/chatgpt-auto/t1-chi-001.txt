For a **2×2 chi-square test of independence**, Cohen’s effect size is **(w)** and the test statistic under the alternative has **noncentrality**
[
\lambda = N w^2,
]
with **df = 1**.

### Step 1) Set the rejection cutoff at (\alpha=0.05)

For a chi-square test (right-tailed),
[
c = \chi^2_{1,,0.95} \approx 3.841,
\quad \sqrt{c}\approx 1.959.
]

### Step 2) Link power to the noncentrality (df = 1 trick)

When df = 1, a noncentral chi-square can be written as
[
\chi^2_{1}(\lambda) ;\overset{d}{=}; (Z+\sqrt{\lambda})^2,\quad Z\sim N(0,1).
]
So power is
[
\text{Power} = P!\left((Z+\sqrt{\lambda})^2 > c\right)
\approx P!\left(Z > \sqrt{c}-\sqrt{\lambda}\right),
]
(the left-tail term (P(Z<-\sqrt{c}-\sqrt{\lambda})) is tiny once (\lambda) is moderate).

Set power = 0.80:
[
0.80 \approx 1-\Phi(\sqrt{c}-\sqrt{\lambda})
;\Rightarrow;
\Phi(\sqrt{c}-\sqrt{\lambda})=0.20.
]
The 20th percentile of (N(0,1)) is (z_{0.20}\approx -0.842), so
[
\sqrt{c}-\sqrt{\lambda} = -0.842
;\Rightarrow;
\sqrt{\lambda} = \sqrt{c}+0.842 \approx 1.959+0.842 = 2.801.
]
Thus
[
\lambda \approx 2.801^2 \approx 7.85.
]

### Step 3) Solve for (N)

[
\lambda = N w^2 \Rightarrow N = \frac{\lambda}{w^2}
= \frac{7.85}{0.3^2}
= \frac{7.85}{0.09}
\approx 87.2.
]

### Result

You need **about (N \approx 88) total observations** (rounding up), for **80% power** to detect **(w=0.3)** at **(\alpha=0.05)** in a **2×2** chi-square independence test.

*(Rule of thumb check: that’s roughly ~44 per group if it’s naturally split 50/50, but the test is about the full 2×2 table, so the key is the **total (N)**.)*
