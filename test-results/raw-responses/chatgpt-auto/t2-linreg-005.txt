To test whether **one specific predictor** adds meaningfully in a multiple regression with **8 total predictors** (so you’re testing an **incremental / partial effect**), the standard power calculation uses the **extra sum-of-squares F test**:

* Numerator df: (u = 1) (the one predictor you’re testing)
* Denominator df: (v = N - k - 1 = N - 8 - 1 = N - 9)

You gave **partial (f^2 \approx 0.053)**, where for an incremental test
[
f^2=\frac{\Delta R^2}{1-R^2_{\text{full}}}.
]

### Noncentral F setup

For this “added predictor” test, the noncentrality parameter is
[
\lambda = f^2,(v+u+1).
]
Substitute (v=N-9), (u=1):
[
\lambda = f^2\big((N-9)+1+1\big)= f^2,(N-7).
]

Power is then
[
\text{Power} = P\left(F_{u,v,\lambda} > F_{1-\alpha;u,v}\right),
]
with (\alpha=0.05).

### Result

Solving this for **90% power** with (f^2=0.053), (u=1), (k=8):

* **Minimum (N \approx 208)** total subjects.

(At (N=207), power is ~0.900; at (N=208), power is ~0.901.)

### Practical add-on

If you expect attrition/missingness, inflate:

* 10% loss: (208 / 0.90 \approx 232)
* 15% loss: (208 / 0.85 \approx 245)

So: **208 analyzable subjects** (or **~232–245 enrolled** if you anticipate 10–15% loss).
