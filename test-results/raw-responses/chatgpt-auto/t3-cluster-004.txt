You’ve fixed the number of clusters (**12 clinics per arm**), so the only lever is the **cluster size** (m) (patients per clinic). A standard planning approximation is:

1. **Compute the individually-randomized required sample size per arm**, then
2. **Adjust for clustering** via the design effect (DE = 1+(m-1)\rho), where (\rho) is the ICC.

---

## 1) Individually randomized trial (two proportions)

Control (p_C = 0.30), Treatment (p_T = 0.20), difference (\Delta = 0.10).
Two-sided (\alpha=0.05 \Rightarrow z_{1-\alpha/2}=1.96).
Power (80% \Rightarrow z_{1-\beta}=0.842).
(\bar p = (0.30+0.20)/2 = 0.25).

Approximate per-arm sample size:
[
n_{\text{ind}} \approx
\frac{\left[
z_{1-\alpha/2}\sqrt{2\bar p(1-\bar p)} + z_{1-\beta}\sqrt{p_C(1-p_C)+p_T(1-p_T)}
\right]^2}{\Delta^2}
]

Plugging in:

* (\sqrt{2\bar p(1-\bar p)}=\sqrt{2(0.25)(0.75)}=\sqrt{0.375}=0.612)
* (\sqrt{p_C(1-p_C)+p_T(1-p_T)}=\sqrt{0.21+0.16}=\sqrt{0.37}=0.608)

[
n_{\text{ind}} \approx \frac{(1.96\cdot 0.612 + 0.842\cdot 0.608)^2}{0.10^2}
\approx \frac{(1.199 + 0.512)^2}{0.01}
\approx \frac{1.711^2}{0.01}
\approx 293
]

So you’d want about **293 effective patients per arm** if individually randomized.

---

## 2) Convert to cluster trial with fixed (k=12) clinics/arm and ICC (\rho=0.02)

With equal cluster sizes (m), the **effective sample size per arm** is:
[
n_{\text{eff}} = \frac{k m}{DE} = \frac{k m}{1+(m-1)\rho}
]

Set (n_{\text{eff}} \approx n_{\text{ind}}) and solve for (m):
[
\frac{k m}{1+(m-1)\rho} = n_{\text{ind}}
]

This rearranges cleanly to:
[
m = \frac{n_{\text{ind}}(1-\rho)}{k - n_{\text{ind}}\rho}
]

Plug in (n_{\text{ind}}=293), (k=12), (\rho=0.02):
[
m \approx \frac{293(0.98)}{12 - 293(0.02)}
= \frac{287.1}{12 - 5.86}
= \frac{287.1}{6.14}
\approx 46.8
]

### ✅ Answer

**Plan for about 47 patients per clinic** (round up).

That implies totals:

* **Per arm:** (12 \times 47 = 564) patients
* **Overall:** (24 \times 47 = 1128) patients

---

### Quick practical note

If you expect missing outcomes / dropouts, inflate (m) by dividing by expected retention (e.g., 90% retention → (47/0.9 \approx 53) per clinic).
